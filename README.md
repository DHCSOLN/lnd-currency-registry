# LND Currency Registry

Machine-readable currency registry for LND (Loc Nation Dollar)

Overview

The LND Currency Registry provides a standardized reference for developers, accounting platforms, payment systems, ERP software, treasury systems, financial applications, and interoperability projects seeking to support LND.

Currency Information

| Field                  | Value                            |
| ---------------------- | -------------------------------- |
| Currency Code          | LND                              |
| Currency Name          | Loc Nation Dollar                |
| Numeric Code           | 666                              |
| Minor Unit             | 2                                |
| Minor Unit Description | 2 decimal places                 |
| Type                   | Currency                         |
| Category               | Global-Currency           |
| Issuer                 | State of Loc Nation Central Bank |

Jurisdiction Information

| Field   | Value                       |
| ------- | --------------------------- |
| Name    | State of Loc Nation G.P.B.C |
| Alpha-2 | LN                          |
| Alpha-3 | SOL                         |
| Numeric | 666                         |

Machine Readable Registry

The official registry file is:

```text
currency.json
```

Software systems may reference this file to retrieve LND metadata and populate currency configuration tables, dropdown selections, accounting systems, treasury applications, payment gateways, reporting tools, and financial software integrations.

Example Display

```text
LND – Loc Nation Dollar
```

Registry Purpose

The registry is intended to provide:

* Currency identification
* Developer interoperability
* Software integration reference
* Accounting and treasury system configuration
* Technical implementation guidance
